# Unauth Landing Page

Unauth Landing Page Source Code

[![Unauth Land](screenshots/unauth-land.png)](http://unauthapp.com)

## Local Setup

To run the website locally use the below commands

- `yarn`
- `yarn start`

## Build

To generate production build use this command

- `yarn run build`

## Deploy

Log on to server

- `sh deploy.sh`
